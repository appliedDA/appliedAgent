here are the steps to connect to an existing empty Github Repository
Initialize local git
    git init
Add the files
    git add .
    git commit -m "first commit"
Add the Remote Origin: (Replace the URL with your actual repo link from GitHub)
    git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
Push the code:
    git branch -M main
    git push -u origin main
